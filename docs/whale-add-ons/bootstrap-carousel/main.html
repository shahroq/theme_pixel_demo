concrete5 Bootstrap Carouselis a responsive slideshow component for cycling through elements, like a carousel.

Features & Options:

  • Responsive: Works in all major browsers and devices
  • Slider generator at Dashboard
  • Theme Switcher
  • Add Header/Description/Link (External, Internal)/Button/Bg Color for every slide
  • Slider Interval: The amount of time to delay between automatically cycling an item
  • Pause: Pauses the cycling of the carousel on mouse enter and resumes the cycling of the carousel on mouse leave.
  • Wrap: Whether the carousel should cycle continuously or have hard stops.
  • Keyboard: Whether the carousel should react to keyboard events.
  • Carousel Height
  • Carousel Bg Color
  • Show Controls: Display Next/Previous icons
  • Show Indicators: Display bottom indicator icons
  • Advanced Carousel Design: Add CSS for each item separately
